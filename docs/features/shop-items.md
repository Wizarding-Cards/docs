## Shopping Strategy

**Priority Purchases:**
1. **Rapid Reclamation Remedy** (emergency fund: keep 2-3 for streak protection)
2. **Resource Caches** (when you need resources to craft specific cards)
3. **Frugality Flasks** (when crafting expensive Legendary/Mythic cards)
4. **Rapid Bounty Draught** (to maximise cards per claim session)

## How to Shop

```
/shop
```

1. Browse items (use arrows)
2. Select from dropdown
3. Confirm purchase
4. Item goes to inventory
5. Use via `/profile` ‚Üí Inventory
